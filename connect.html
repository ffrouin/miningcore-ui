Miner Configuration

This is the basic guide how to setup your miner to this pool. Please select the currency in the menu in order the pool details (port, diff, fee) to refresh. Default pool details displayed is Monero ($XMR).

Mine Monero (XMR) with CPUs, GPUs and XMRig

Mine Litecoin (LTC) with ASICs FutureBit Moonlander 2 and BFGminer

Mine Vertcoin (VTC) with NVidia devices and CCMiner

Mine Bitcoin (BTC) with NVidia devices and CCMiner

All miner software will allmost require the same options to connect successfully :

-a <algo> -o stratum+tcp://andromede.linuxtribe.fr:<port> -u <your_wallet_address>

You don't need a password to connect to stratum server. You will appear in the list of active miners after you start find some share and the server to refresh the dashboard.


Support
